# Hasher

Hasher is a Clojure library for manipulating fixed-size hash maps using arrays.

Linear probing is employed for hash collisions.

## Setup

```
$ brew install leiningen
$ git clone https://github.com/nspotrepka/hasher.git
$ cd hasher
```

## Playground

```
$ lein repl
```

## Testing

```
$ lein test
```
